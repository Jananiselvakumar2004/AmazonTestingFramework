## 📌 Project Overview

Amazon Testing Framework is a Selenium automation project developed using **Java 21**, **Selenium WebDriver**, **TestNG**, and **Maven**.

The framework automates Amazon product searches, extracts product details, captures screenshots, generates execution reports, and stores extracted information using a data-driven testing approach.

## ▶️ How to Run

1. Clone this repository.
2. Open the project in IntelliJ IDEA.
3. Update Maven dependencies.
4. Run `testng.xml`.
