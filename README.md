# Amazon Testing Framework

## Project Overview

Amazon Testing Framework is a Selenium Automation Framework developed using Java, Selenium WebDriver, TestNG, and Maven. It automates product search on Amazon, extracts product details, captures screenshots, generates reports, and stores the results in CSV files.

## Features

- Selenium WebDriver Automation
- Java 21
- TestNG Framework
- Page Object Model (POM)
- CSV Data-Driven Testing
- Screenshot Capture
- Extent Reports
- Retry Analyzer
- Log4j Logging
- Maven Project Structure

## Technologies Used

- Java 21
- Selenium WebDriver
- TestNG
- Maven
- OpenCSV
- Apache POI
- Extent Reports
- Log4j
- WebDriverManager

## How to Run

1. Clone the repository.
2. Open in IntelliJ IDEA.
3. Update Maven dependencies.
4. Run testng.xml.
